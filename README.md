# TranslateZView

Change the zIndex 

(Useful to change z Index of view that you have no control) (RecyclerListview)

android: api 21+   
ios : 10.0+   
web : zIndex


## Getting started

`$ npm install react-native-translatezview --save`  

yarn:

`$ yarn add react-native-translatezview`


## Usage
```javascript
import ZView from 'react-native-translatezview';
```
